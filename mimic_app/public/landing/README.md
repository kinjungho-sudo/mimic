# 랜딩페이지 데모 영상

이 폴더에 아래 파일명으로 mp4를 넣으면 랜딩페이지 "제품 투어" 섹션의 목업이 실제 영상으로 자동 교체됩니다.
(파일이 없으면 코드로 그린 화면 목업이 대신 표시됩니다 — 깨지지 않음)

| 파일명 | 찍을 내용 | 권장 길이 |
|---|---|---|
| `record.mp4` | 크롬 확장 켜고 클릭 → 자동 캡처되는 모습 (캡처 토스트 포함) | 10~20초 |
| `edit.mp4` | 에디터에서 화살표/박스/모자이크 추가 + 영역 드래그 줌인 | 10~20초 |
| `edu.mp4` | 교육 자료 모드 녹화 → AI 설명 자동 생성되는 모습 | 10~20초 |
| `guideme.mp4` | Guide Me 오버레이가 실제 페이지 위에서 단계 안내하는 모습 | 10~20초 |
| `share.mp4` | 공유 링크 복사 + PDF/PPTX 내보내기 + 브랜딩 설정 | 10~20초 |

권장 사양: 1280×800 내외(가로형), 무음(자동재생을 위해 muted 처리됨), 루프 재생됨, 파일당 5MB 이하 권장.
