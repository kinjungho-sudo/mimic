# 첫 사용자 시작 튜토리얼 계획

## 문제

첫 사용자는 빈 워크스페이스에 진입한 뒤 `화면 녹화`, `직접 편집`, `새 플레이북` 중 무엇부터 해야 하는지 알기 어렵다.

## 목표

첫 사용자가 설명을 읽는 데서 끝나지 않고, 실제 녹화 흐름을 시작해 첫 매뉴얼 편집 화면까지 도달하도록 안내한다.

## MVP 범위

1. 기능 도입 이후 가입했고 개인 워크스페이스에 매뉴얼이 없으며 튜토리얼 이력이 없는 사용자에게 시작 카드를 표시한다.
2. `첫 매뉴얼 만들기`를 누르면 기존 `RecordingModal`을 연다.
3. 튜토리얼은 언제든 건너뛸 수 있고, 빈 상태에서 다시 시작할 수 있다.
4. 튜토리얼을 시작한 사용자가 녹화를 마치고 편집 화면에 들어오면 완료 안내를 표시한다.
5. 상태는 사용자 ID가 포함된 브라우저 저장소 키에 보관한다.

## 제외 범위

- 랜딩페이지 데모 및 영상
- 플레이북 튜토리얼
- Recorder 녹화 로직 변경
- Supabase 스키마 변경

## 완료 기준

- 기존 사용자와 팀 워크스페이스에는 자동 노출되지 않는다.
- 첫 사용자에게 주요 행동이 하나만 강조된다.
- 건너뛰기 후 강제로 다시 열리지 않는다.
- 녹화 완료로 열린 편집기에서 첫 매뉴얼 완성을 확인할 수 있다.
- 키보드 포커스와 모바일 레이아웃을 지원한다.
