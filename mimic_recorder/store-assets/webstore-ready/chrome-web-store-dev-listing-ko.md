## 스토어 등록정보

### 확장 프로그램 이름

Parro Recorder DEVELOPMENT BUILD

### 간단한 설명 (132자 이하)

웹에서 수행한 클릭과 입력 흐름을 화면과 함께 기록해 단계별 매뉴얼로 만드는 Parro 개발 테스트용 레코더입니다.

### 상세 설명

THIS EXTENSION IS FOR BETA TESTING.

Parro Recorder DEVELOPMENT BUILD는 사용자가 웹에서 수행하는 작업 흐름을 단계별로 기록하고, 편집 가능한 업무 매뉴얼로 변환하는 Parro의 개발 테스트용 Chrome 확장 프로그램입니다.

사용자가 녹화를 시작한 동안에만 선택한 탭·창·화면을 캡처합니다. 클릭, 입력, 페이지 이동 등의 작업 정보를 화면 이미지와 함께 기록하고 Parro dev 서비스로 전송하여 단계별 매뉴얼을 생성합니다.

주요 기능

- 사용자가 선택한 탭·창·화면 녹화
- 클릭, 입력 및 페이지 이동을 단계별로 기록
- 단계별 화면 캡처와 설명 자동 생성
- 비밀번호 입력값 제외 및 민감정보 영역 블러 처리 지원
- 선택적 마이크 음성 설명 녹음
- 생성된 매뉴얼 확인 및 편집
- 전체 페이지 이미지 캡처 및 다운로드
- 생성된 가이드를 웹페이지에서 단계별로 실행

이 빌드는 Parro dev 환경의 기능 검증과 제한된 테스터 피드백 수집을 위한 버전입니다. 운영 환경에서의 일반 사용을 위한 버전이 아닙니다.

사용자가 직접 녹화를 시작하지 않으면 화면이나 웹 활동을 기록하지 않습니다. 녹화는 언제든지 일시정지하거나 종료할 수 있습니다.

## 개인정보 보호 관행

### 단일 목적 설명

사용자가 명시적으로 녹화를 시작한 동안 선택한 웹 화면과 작업 상호작용을 캡처하여 Parro에서 확인하고 편집할 수 있는 단계별 업무 매뉴얼을 생성합니다.

### 권한 사용 사유

#### activeTab

사용자가 녹화를 시작한 현재 탭을 식별하고, 해당 탭의 화면과 작업 흐름을 캡처하기 위해 사용합니다.

#### tabs

녹화 대상 탭의 활성 상태, 페이지 이동, URL 및 제목을 확인하고, 녹화 중 탭 전환을 추적하며, 생성 완료 후 Parro 매뉴얼 편집 페이지를 열기 위해 사용합니다.

#### desktopCapture

사용자가 Chrome의 화면 선택 창에서 직접 선택한 탭·창·화면을 녹화하고 단계별 이미지를 생성하기 위해 사용합니다. 사용자의 선택과 승인 없이 화면 캡처를 시작하지 않습니다.

#### windows

캡처 대상 창을 식별하고 포커스를 관리하며, 선택적 음성 설명 기능에 필요한 마이크 권한 요청 창을 표시하기 위해 사용합니다.

#### storage

확장 프로그램 설정, 로그인 연동 토큰, 녹화 상태, 단계 정보 및 전송 전 임시 캡처 데이터를 브라우저 로컬 저장소에 보관하기 위해 사용합니다.

#### downloads

사용자가 전체 페이지 캡처 기능을 실행했을 때 생성된 PNG 이미지를 사용자의 다운로드 폴더에 저장하기 위해 사용합니다.

#### sidePanel

녹화 시작·일시정지·종료, 단계 확인 및 설정 기능을 Chrome 사이드 패널에서 제공하기 위해 사용합니다.

#### scripting

사용자가 녹화를 시작한 웹페이지에서 클릭·입력·페이지 이동을 감지하고, 민감정보 블러 처리 및 단계별 안내 UI를 제공하는 로컬 패키지 스크립트를 실행하기 위해 사용합니다.

#### offscreen

사용자가 승인한 화면 스트림에서 단계별 이미지를 추출하고, 선택적으로 승인한 마이크 음성을 녹음하기 위해 백그라운드 오프스크린 문서를 사용합니다.

#### 호스트 권한 (`<all_urls>`)

Parro Recorder는 사용자가 선택한 다양한 웹사이트에서 작업 절차를 기록하는 범용 매뉴얼 제작 도구입니다. 녹화가 시작된 웹페이지에서 클릭·입력·페이지 이동을 감지하고 화면을 캡처하기 위해 웹사이트 접근 권한이 필요합니다. 확장 프로그램은 사용자가 직접 녹화를 시작한 세션에서만 수집한 정보를 매뉴얼 생성에 사용합니다.

#### parro-guide-dev.vercel.app 및 Parro 서비스 도메인

확장 프로그램과 Parro 웹앱의 계정 연동, 녹화 단계 저장, 매뉴얼 생성 완료 처리 및 생성된 매뉴얼 열기를 위해 사용합니다.

#### Supabase 도메인

사용자가 녹화한 단계별 화면 이미지와 선택적 음성 파일을 Parro의 저장소에 안전하게 업로드하기 위해 사용합니다.

### 원격 코드 사용

아니요. 원격 코드를 실행하지 않습니다.

확장 프로그램 실행 코드는 업로드된 패키지에 포함되어 있습니다. Parro 서버 및 Supabase와 HTTPS API 통신을 수행하지만, 원격 JavaScript 또는 WebAssembly 코드를 다운로드하여 실행하지 않습니다.

### 수집·처리하는 데이터 유형

대시보드에 표시되는 실제 선택지와 대조하여 다음 항목을 공개합니다.

- 인증 정보: Parro 계정과 확장 프로그램을 연동하는 세션 토큰
- 웹 기록: 녹화 중인 페이지의 URL과 제목
- 사용자 활동: 사용자가 녹화 중 수행한 클릭, 입력 및 페이지 이동
- 웹사이트 콘텐츠: 사용자가 선택한 화면의 스크린샷, 요소 정보 및 입력 내용
- 사용자 생성 콘텐츠: 사용자가 선택적으로 녹음한 음성 설명
- 개인 식별 정보: Parro 계정 제공과 연동에 필요한 이메일, 이름 또는 계정 식별자

비밀번호 입력값은 수집하지 않습니다. 일반 입력 내용은 매뉴얼 단계 설명을 만들기 위해 처리될 수 있으므로, 민감한 정보가 포함된 페이지를 녹화하지 않도록 사용자에게 안내합니다.

### 제한적 사용 인증

대시보드에서 다음 내용을 모두 사실대로 인증합니다.

- 수집한 데이터는 Parro Recorder의 단일 목적과 사용자 대면 기능 제공 또는 개선에만 사용합니다.
- 데이터를 맞춤형 광고, 신용평가 또는 데이터 판매에 사용하지 않습니다.
- 서비스 제공에 필요한 처리업체 외에는 사용자 데이터를 판매하거나 이전하지 않습니다.
- 사용자가 요청한 지원, 보안 조사 또는 법적 의무 이행 등의 허용된 경우를 제외하고 사람이 사용자 데이터를 읽도록 허용하지 않습니다.

## 심사자 테스트 안내

### 테스트 준비

이 확장 프로그램은 Parro 계정 로그인이 필요합니다. 심사용 테스트 계정의 이메일과 비밀번호를 대시보드의 별도 자격 증명 필드에 제공합니다. 자격 증명은 공개 설명란에 입력하지 않습니다.

### 테스트 절차

1. https://parro-guide-dev.vercel.app 에 접속하여 제공된 심사용 계정으로 로그인합니다.
2. Parro 홈 화면에서 Recorder 연결을 시작하고 설치된 확장 프로그램과 계정을 연동합니다.
3. 테스트할 일반 웹페이지를 새 탭에서 엽니다.
4. Chrome 도구 모음에서 Parro Recorder를 열어 사이드 패널을 표시합니다.
5. `녹화 시작`을 누릅니다.
6. Chrome이 화면 선택 창을 표시하면 테스트할 탭을 선택하고 공유를 승인합니다.
7. 선택한 페이지에서 버튼 클릭, 링크 이동 또는 일반 텍스트 입력을 수행합니다.
8. 사이드 패널에서 기록된 단계와 화면 미리보기를 확인합니다.
9. `녹화 종료`를 누릅니다.
10. Parro dev의 매뉴얼 편집 페이지가 열리고, 캡처된 단계가 순서대로 생성되는지 확인합니다.

### 선택적 마이크 테스트

1. Recorder 설정에서 음성 설명 기능을 켭니다.
2. 표시되는 권한 창에서 마이크 사용을 허용합니다.
3. 녹화 중 음성으로 설명한 후 녹화를 종료합니다.
4. 생성된 매뉴얼에 음성 처리 결과가 반영되는지 확인합니다.

### 개인정보 보호 동작

- 녹화는 사용자가 `녹화 시작`을 눌러야 시작됩니다.
- 비밀번호 입력란의 값은 기록하지 않습니다.
- 녹화 중인 캡처와 작업 정보만 매뉴얼 생성을 위해 전송합니다.
- 마이크는 사용자가 음성 기능을 켜고 권한을 허용한 경우에만 사용합니다.

## 제출 전 필수 확인

- manifest 이름을 `Parro Recorder DEVELOPMENT BUILD`로 유지
- manifest 설명의 깨진 한글 인코딩 수정
- dev 등록을 운영 등록으로 오인하게 만드는 문구가 없는지 확인
- 실제 사용하지 않는 권한 제거
- 개인정보처리방침이 로그인 없이 공개 접근되는지 확인
- 개인정보처리방침에 화면, URL, 입력 정보, 음성, 계정 토큰 및 처리업체를 명시
- 심사용 테스트 계정 준비
- 최종 ZIP 재생성 및 업로드 전 로컬 설치 테스트
