## 단일 목적 설명 (심사용 — 권한 사유 포함)

웹페이지에서 사용자의 클릭 동작을 단계별로 캡처하여 매뉴얼을 자동 생성하는 도구입니다. 녹화 활성화 중에만 스크린샷을 캡처하며, 사용자의 명시적 동작(클릭/입력)에 의해서만 작동합니다. 캡처 시 비밀번호·이메일·전화번호 등 개인식별정보(PII)를 자동으로 감지하여 블러 처리한 뒤 저장하므로 민감정보가 스크린샷에 노출되지 않습니다.

권한 사용 사유:
- `<all_urls>`: 사용자가 어떤 웹사이트든 자유롭게 녹화할 수 있어야 합니다.
- `tabs`: 녹화 대상 탭을 식별하고 화면을 캡처하기 위해 필요합니다.
- `scripting`: 페이지 로드 완료 여부를 확인한 뒤 캡처하여 검은 화면을 방지합니다.
- `storage`: 녹화 세션 및 설정을 로컬에 저장합니다.
- `sidePanel`: 녹화 컨트롤 및 캡처된 스텝 목록을 사이드패널에 표시합니다.
- `downloads`: 녹화된 스텝을 ZIP 파일로 로컬 저장하는 기능에 사용합니다.

## 자세한 설명

MIMIC Recorder

캡처 시작 버튼만 누르고 평소처럼 작업하세요. MIMIC이 모든 단계를 자동으로 캡처하여, 몇 초 만에 스크린샷과 설명이 완성된 매뉴얼을 제공합니다.

- 시간 절약: 화면을 캡처하고 설명을 입력하는 번거로운 작업 없이, MIMIC이 모든 단계를 자동으로 기록하고 깔끔하게 정리해드립니다.
- 반복 질문 제로: 팀원이 같은 걸 또 물어볼 때, 링크 하나로 끝내세요. 한 번 만든 매뉴얼이 계속 일합니다.
- 즉시 공유: 생성된 매뉴얼은 링크 공유, PDF 저장, 또는 슬랙·노션·이메일에 붙여넣어 바로 배포할 수 있습니다.

MIMIC Recorder 주요 기능
- 자동 단계 캡처: 브라우저에서 작업하면 MIMIC이 클릭·입력·화면 전환을 자동으로 감지해 단계별 가이드를 만듭니다.
- AI 설명 자동 생성: 각 단계마다 AI가 자연스러운 설명을 자동으로 작성합니다. 직접 수정도 가능합니다.
- PII 자동 보호: 캡처 시 비밀번호·이메일·전화번호·카드번호 등 개인식별정보를 자동으로 감지하여 블러 처리합니다. 민감정보가 스크린샷에 노출되지 않아 안전하게 매뉴얼을 공유할 수 있습니다.
- 스마트 편집: 단계별 스크린샷을 편집하고, 강조 표시를 추가하며, 민감한 정보를 안전하게 가릴 수 있습니다.
- 다양한 공유 옵션: 클릭 한 번으로 공유 링크를 생성하거나, 슬랙·노션·이메일 등 자주 쓰는 곳에 바로 공유하세요.

이런 분들께 추천합니다
MIMIC Recorder는 반복적인 설명이 필요한 모든 분들께 유용합니다:

- 운영 및 IT: 사내 시스템 사용법, 반복 업무 절차를 표준화하고 누구나 따라할 수 있게 문서로 만드세요.

- 영업 및 마케팅: 툴 사용법, 제안서 작성 프로세스를 팀 전체가 동일하게 따를 수 있도록 문서화하세요.
