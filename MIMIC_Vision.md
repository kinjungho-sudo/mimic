# MIMIC — 제품 비전 브리핑

> 이 문서는 AI 어시스턴트가 MIMIC 서비스의 본질과 장기 방향을 정확히 이해하기 위한 컨텍스트 문서입니다.

---

## 한 줄 정의

**MIMIC은 사람의 업무 패턴을 학습해, AI가 그 업무를 스스로 따라할 수 있게 만드는 플랫폼이다.**

---

## 지금 단계 (MVP)

겉으로 보이는 것: **AI 인터랙티브 매뉴얼 자동 생성 도구**

- Chrome 확장 프로그램으로 화면 캡처
- AI가 각 단계의 제목·설명 자동 생성
- 공유 링크로 단계별 인터랙티브 뷰어 제공
- 경쟁사: Tango(미국), Guidde(미국), 스텝하우(한국)

지금은 **"사람이 읽는 매뉴얼"** 을 만드는 것처럼 보인다.
하지만 이건 시작일 뿐이다.

---

## 진짜 목표 (장기 비전)

**사람이 한 번 한 업무를 AI가 스스로 따라하는 것.**

매뉴얼을 만드는 과정에서 사람의 클릭, 이동, 입력 패턴이 데이터로 쌓인다.
이 데이터가 AI 에이전트의 학습 인풋이 된다.
결국 AI가 그 업무를 대신 수행하는 자동화 인프라로 진화한다.

```
지금:  사람이 클릭 → MIMIC이 캡처 → 사람이 보는 매뉴얼 생성
미래:  사람이 클릭 → MIMIC이 학습 → AI가 그 클릭을 대신 수행
```

**MIMIC(흉내쟁이)** 이라는 이름이 이 본질을 담고 있다.

---

## 경쟁사와의 차이

| | 스텝하우 / Tango / Guidde | **MIMIC** |
|---|---|---|
| 최종 산출물 | 사람이 읽는 문서 | AI가 학습하는 업무 패턴 데이터 |
| 목표 | 문서화 도구 | 업무 자동화 인프라 |
| 게임 | 완성형 제품 경쟁 | 다른 게임 |

스텝하우, Tango는 훌륭한 서비스다. 하지만 그들의 끝은 **문서**다.
MIMIC의 끝은 **자동화**다.

---

## 대외 포지셔닝 원칙

| 상황 | 말할 것 |
|------|---------|
| 일반 발표 / 마케팅 | 매뉴얼 자동화, 사람을 위한 효율화 플랫폼 |
| "경쟁사랑 뭐가 달라요?" 질문 | 비전 카드 꺼내기 |
| 투자자 피칭 | 데이터 수집 레이어 → AI 자동화 인프라 진화 스토리 |

**비전은 먼저 꺼내는 카드가 아니다. 질문받을 때 꺼내는 카드다.**

지금 MVP 단계에서 "AI 자동화 인프라"를 전면에 내세우면:
- 법적 리스크 우려
- 실현 가능성 의심
- 지금 당장의 가치가 희석됨
